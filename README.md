# TOP Foundation Landing Page Project

## Project Overview

Build a complete static web page based on provided design mockups (layout + style guide). The goal is to recreate the overall structure and appearance—not pixel-perfect accuracy—using HTML and CSS.

## Objectives

- Recreate the webpage layout from the design images  
- Practice structuring content with HTML before styling with CSS  
- Use CSS (especially Flexbox) to position elements  
- Work section-by-section (4 main sections + footer)  
- Keep code organized in a single stylesheet  

## Guidelines

- Focus on visual similarity, not exact spacing or measurements  
- It’s okay to look things up—research is part of the process  
- Minor design details may require independent problem-solving  
- You can customize content, images, colors, and fonts  

## Images & Credits

Use only free-to-use images (e.g., Pexels, Pixabay, Unsplash) and credit creators in the README.

## Workflow

- Initialize a GitHub repository before starting  
- Commit changes frequently with clear messages  
- Build one section at a time (start from the top)  
- Add all HTML content first, then apply CSS styling  
- No need to make the page responsive  

## Deployment

Once complete, deploy the project using GitHub Pages to share it online.
